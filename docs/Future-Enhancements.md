# Future Enhancements

# Overview

This document outlines planned improvements for the Active Directory Domain Controller Lab. While the current project successfully demonstrates the deployment of a Windows Server 2022 Domain Controller, enterprise Active Directory environments typically include additional services, client systems, automation, and security controls.

These enhancements will be implemented as future phases of the lab and documented within this repository.

---

# Phase 2 – Windows 11 Domain Client

## Objective

Deploy a Windows 11 Pro virtual machine and join it to the **griffith.local** domain.

### Planned Tasks

- Install Windows 11 Pro
- Configure Host-Only networking
- Configure DNS
- Join the Active Directory domain
- Verify domain authentication
- Log in using domain credentials

### Skills Demonstrated

- Domain Join
- Windows Client Administration
- Enterprise Authentication
- DNS Troubleshooting
